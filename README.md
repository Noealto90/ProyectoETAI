# ProyectoETAI

## Descripción
ProyectoETAI es un sistema de gestión de reservas y equipos para laboratorios educativos. Permite a los usuarios reservar espacios, reportar daños y gestionar equipos de manera eficiente.

## Características
- **Autenticación de Usuarios**: Inicio de sesión y registro de usuarios con roles específicos (superAdmin, estudiante, profesor, administrativo).
- **Gestión de Reservas**: Reserva de espacios en laboratorios para diferentes actividades.
- **Gestión de Equipos**: Añadir, actualizar y restaurar equipos en los laboratorios.
- **Reportes de Daños**: Reportar y gestionar daños en los equipos.
- **Generación de Informes**: Generar informes detallados del uso de laboratorios.
